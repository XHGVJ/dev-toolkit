# 🛠️ Dev Toolkit

A curated collection of useful Python developer utilities, scripts, and helper functions.

## 📦 Features

- **String Helpers**: Truncation, slugification, and pattern matching.
- **File Utilities**: Recursive file listing, safe deletion, and metadata extraction.
- **Date Utils**: Timezone conversion and human-readable formatting.
- **CLI Scripts**: Automated cleanup and deployment scripts.

## 🚀 Getting Started

```bash
git clone https://github.com/XHGVJ/dev-toolkit.git
cd dev-toolkit
pip install -r requirements.txt
```
